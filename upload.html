🧪 众包课程上传

第一阶段 MVP：把结构化课程包保存到浏览器本地。主网页输入课程码后，会动态接入同一个阅读器。

课程包 JSON

上传路径

1. 复制 skill 生产课程
每节必须有标题术语、阿波 hook、正文、UX、after、terms、close。
2. 按 JSON 格式上传
课程码 `code` 是隐藏查看入口。小范围测试时，保存到同源 localStorage 即可。
3. 主网页输入课程码
主站动态读取课程包，转成 `CHAPTERS['lab:CODE']`，复用现有阅读器。
4. 归档
好作品后续改 `status: official` 并加入正式课程索引。
