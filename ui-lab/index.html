2训练 = 看例子 → 猜 → 对答案 → 调整

训练就是不停重复一个循环：喂一张图 → 模型猜答案 → 告诉它对不对 → 它悄悄调整自己 → 再喂下一张。

几万次之后，它就从"瞎猜"变成了"很准"——这不是魔法，是大量重复。
